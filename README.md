# Pikachu
![](./img/Pikachu%20foto.png)

Este projeto apresenta uma tela de boas-vindas simples e vibrante, inspirada no Pokémon mais famoso, o Pikachu!

Com seu design minimalista e dominado pela cor amarela, a página centraliza o foco no Pikachu e usa botões claros como "GET FOR FREE". É um exemplo de layout focado em atrair a atenção do usuário e facilitar a conversão em uma página de destino (landing page) de aplicativo.

## Tecnologias
* HTML
* CSS
* Media Queries
